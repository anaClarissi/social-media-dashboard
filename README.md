## 📋 Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio da **Frontend Mentor**, com o objetivo de construir um dashboard de redes sociais que exibe estatísticas de seguidores e engajamento do Facebook, Twitter, Instagram e YouTube. O grande destaque do desafio é o **alternador de tema (dark/light mode)**, permitindo que o usuário escolha a aparência que preferir.

### ✅ Funcionalidades

- 📱 Layout totalmente responsivo (mobile first)
- 🌗 Alternância entre tema claro e escuro
- 📈 Cards com indicadores de crescimento e queda de seguidores
- 🎨 Cores dinâmicas de acordo com o tema selecionado
- 🧩 Componentização de estilos com Sass

## 🧠 Aprendizados

Durante o desenvolvimento desse projeto, pude evoluir em diversos pontos importantes do front-end:

- 🔄 **Manipulação de temas com `data-theme`**: aprendi a usar atributos customizados no `<body>` combinados com variáveis CSS para trocar paletas de cores inteiras de forma performática, sem duplicar estilos.
- 🎨 **Organização de estilos com Sass**: entendi na prática como estruturar arquivos `.scss` para manter o CSS mais escalável e reutilizável, aproveitando variáveis, aninhamento e imports.
- ⚡ **Fluxo de trabalho com Vite**: configurei um ambiente de desenvolvimento moderno, com hot reload e build otimizado, integrando Bootstrap, Sass e JavaScript no mesmo projeto.
- 📐 **Responsividade orientada a dados reais**: pratiquei a construção de um layout que se adapta fielmente ao design fornecido (mobile, tablet e desktop), respeitando espaçamentos e proporções.
- 🧩 **Separação de responsabilidades no JS**: organizei a lógica de troca de tema em um módulo isolado (`background.js`), reforçando boas práticas de modularização.
- 🚀 **Deploy com Vercel**: aprendi todo o fluxo de publicação de um projeto Vite em produção, do build à disponibilização do link final.
